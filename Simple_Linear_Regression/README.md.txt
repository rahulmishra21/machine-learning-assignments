# Simple Linear Regression

## Overview
This project demonstrates the implementation of **Simple Linear Regression** using Python.
The goal is to model the relationship between a single independent variable and a dependent variable.

## Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Workflow
1. Data loading and inspection
2. Data preprocessing
3. Train-test split
4. Model training using Linear Regression
5. Prediction on test data
6. Visualization of regression results

## Model Used
- Simple Linear Regression (`sklearn.linear_model.LinearRegression`)
